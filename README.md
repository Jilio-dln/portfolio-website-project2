# Portfolio Website – Project 2

Dit is de repository voor mijn portfolio website (project 2) die ik ontwikkel binnen het OPO Websites van de opleiding Graduaat Programmeren.

De website is een responsive versie van mijn portfolio, opgebouwd met Bootstrap 5. Het doel is om een overzicht te geven van mijn groei als student developer en de technieken die ik leer toepassen.

## Inhoud van de website

De website bestaat uit de volgende pagina's:

* **Home** (index.html) – introductie met hero section en overzicht van mijn portfolio
* **CV** (cv.html) – informatie over mijn opleiding, werkervaring, talen en vaardigheden
* **Interesses** (interesses.html) – mijn hobby's en persoonlijke interesses met een Bootstrap carousel
* **Contact** (contact.html) – contactformulier en manieren waarop een werkgever mij kan bereiken

## Gebruikte technologieën

Voor het ontwikkelen van deze website gebruik ik:

* HTML – structuur van de webpagina's
* CSS – eigen styling en aanpassingen bovenop Bootstrap
* Bootstrap 5 – responsive layout en componenten
* JavaScript – interactie en carousel functionaliteit
* Externe carousel plugin – voor de skills carousel op de CV-pagina

## Doel van dit project

Dit project helpt mij om:

* een volledig responsive website te bouwen die werkt op desktop, tablet en smartphone
* te werken met het Bootstrap framework en de bijhorende componenten
* externe plugins te integreren in een webproject
* mijn portfolio verder uit te breiden en te professionaliseren
